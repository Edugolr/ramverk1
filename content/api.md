|Rate Limited	   | No          |

##API
GET

**Ip validering**
`ipvalidatejson/standard?ip=[ip]`

**Resultat**
```
{
    "type": "IPV4",
    "host": "h-148-64.A317.priv.bahnhof.se",
    "ip": "37.123.148.64"
}
```

`ipvalidatejson/location?ip=[ip]`
